# ¿Qué es Python?
Python es un lenguaje de **programación orientado a objetos** (OOP) de **alto nivel** y tipado dinamico, caracterizado por su **sintaxis sencilla** y **facilidad de aprendizaje**.<br>
Todos estos términos tienen un significado, y ahora vamos a explicarlos.

## ¿Qué significa OOP?
La programación orientada a objetos es un estilo de programación en el que el código se hace entorno a datos y objetos en vez de funciones y lógica pura. Un objeto se define como un set de datos, atributos y comportamientos. Ya indagaremos mas en la OOP cuando lleguemos a las clases y objetos como concepto de programación.

## ¿Qué significa 'Alto nivel'?
Cuando decimos que un lenguaje es de bajo nivel, nos estamos refiriendo a que el lenguaje funciona muy cercano al procesador, pudiendo manejar directamente direcciones de memoria y otras cosas que son extremadamente complejas pero que te dan un control sobre la maquina increíble.<br>
Un lenguaje de alto nivel seria lo opuesto, es decir, un lenguaje que abstrae esos conceptos y los hace mas accesibles.

## ¿Qué significa 'tipado dinamico'?
El tipado en un lenguaje de programación es el como se declaran variables en el mismo. En un lenguaje de tipado estático, cuando se declara una variable tienes que declarar el **tipo** de variable que es, lo que impide que luego esa variable se sobre escriba con un tipo distinto del que se uso para declararla.<br>
El tipado dinamico es lo opuesto, es decir, las variables pueden alojar el valor que sea y cambiar de tipo a medida que se ejecuta el programa.

## Sintaxis sencilla
La sintaxis de un lenguaje de programación es el como se escribe el código en dicho lenguaje. Una sintaxis es mas sencilla cuanto mas se parezca a escribir en lenguaje natural, el que nosotros hablamos, y python justamente destaca por esta razón, ya que el código de python se puede leer como si fuera un texto en ingles y se podria entender bastante bien.

## Facilidad de aprendizaje
Debido a su sintaxis sencilla y a ser un lenguaje de alto nivel que abstrae los conceptos complicados de la programación, python es bastante sencillo y sobre todo para los nuevos en la programación.
