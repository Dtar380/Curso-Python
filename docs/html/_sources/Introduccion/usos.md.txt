# ¿Para que se utiliza?
Es importante saber para que se usa Python si vamos a aprender a utilizarlo, ya que al final los lenguajes de programación se inventan siempre desde un punto de vista pragmático, es decir, el lenguaje se orienta a algo practico y util, no algo abstracto.
Python se utiliza principalmente para 3 cosas:
- IA
- Analisis de datos
- Backend Web

Aunque esto sea para lo que mas se usa en la industria, con python se puede hacer casi de todo, y de hecho, nosotros no vamos a estar haciendo ninguna de esas cosas, por que ese tipo de programas son mucho más complejos y requieren conocimientos en otras areas.
