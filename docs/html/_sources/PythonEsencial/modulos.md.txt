# Módulos
Un modulo es una pieza de código creada para poder ser usada mas tarde importandola. Python es un lenguaje que depende de la modularización, hasta la propia instalación básica del lenguaje ya viene con módulos pre-instalados, cosa que con muchos otros lenguajes no pasa. Esto nos da la oportunidad de no tener que escribir dos veces el mismo código, y simplemente poder llamarlo una vez que ya ha sido creado.

Las librerías mas importantes que vienen por defecto integradas en python son:
- os: una libreria que nos permite manejar el sistema operativo desde python.
- math: una libreria con todas las funcines matemáticas y otras relacionadas con números.
- time: una libreria que nos permite manejar tiempos.
- datetime: una libreria que nos permite manejar fechas.
- json: una libreria que nos permite manejar archivos y datos de tipo JSON.

En caso de que necesitéis revisar como funcionan estas y otras librerías integradas en el lenguaje de python o como se usan ciertas funciones siempre podéis revisar la [documentación oficial de python](https://docs.python.org/3.12/search.html).
