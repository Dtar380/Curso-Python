# Funciones
No vamos a indagar mucho en lo que es una función todavía, pero antes de comenzar tendremos que saber las funciones mas básicas del lenguaje python, ya que como hemos dicho anteriormente, python es un lenguaje de alto nivel, esto quiere decir que todo lo que es bajo nivel (que siempre es necesario) nos lo dan directamente hecho, pero tenemos que saber que funciones nos van a permitir hacer ciertas cosas.

Para llamar a una funcion en python se hace de la siguiente manera:
```{code-block} python
funcion(argumentos)
```

En [esta](https://www.w3schools.com/python/python_ref_functions.asp) pagina podréis encontrar todas las funciones básicas de python.

Aparte de estas, hay otras cuantas que no están como tal integradas en el lenguaje, pero si en los módulos que vienen preinstalados.
